# Common Network Ports
| Port | Protocol/Service | Transport Protocol | Purpose                        |
| ---- | ---------------- | ------------------ | ------------------------------ |
| 20   | FTP (Data)       | TCP                | Transfers FTP data             |
| 21   | FTP (Control)    | TCP                | FTP connection and commands    |
| 22   | SSH              | TCP                | Secure remote login            |
| 23   | Telnet           | TCP                | Remote login (unencrypted)     |
| 25   | SMTP             | TCP                | Sending emails                 |
| 53   | DNS              | TCP/UDP            | Domain name resolution         |
| 67   | DHCP Server      | UDP                | Assigns IP addresses           |
| 68   | DHCP Client      | UDP                | Receives IP configuration      |
| 69   | TFTP             | UDP                | Trivial File Transfer Protocol |
| 80   | HTTP             | TCP                | Web browsing                   |
| 110  | POP3             | TCP                | Receiving emails               |
| 123  | NTP              | UDP                | Time synchronization           |
| 143  | IMAP             | TCP                | Email retrieval                |
| 161  | SNMP             | UDP                | Network monitoring             |
| 162  | SNMP Trap        | UDP                | SNMP alerts and notifications  |
| 389  | LDAP             | TCP/UDP            | Directory services             |
| 443  | HTTPS            | TCP                | Secure web browsing            |
| 445  | SMB              | TCP                | File and printer sharing       |
| 3389 | RDP              | TCP                | Remote Desktop Protocol        |
